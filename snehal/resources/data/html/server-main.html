Welcome

This is a Database Wiki: a system for creating and sharing structured data, using a familiar wiki interface. Click on any of the database names to the left to browse data.

The menus along the top of the page allow:

  • editing (creating new records, changing existing records, or deleting)
  • viewing (changing to another database or returning to this server home page)
  • editing the layout, style sheet or template for a page

All editing actions are recorded, and all prior versions of the data are available. The time-machine links along the top of the page provide access to previous versions and allow comparing versions. Also, click on "comments" or "history" on a data page to see existing comments on a data object, add your own comments, or see the history of operations applied to the object or its subobjects. The presence of comments on parts of data is also highlighted by a "word balloon" ()

Editing and viewing some pages or data requires logging in. This demonstration system doesn't yet have the capability to add new users on-the-fly. For this demonstration, anyone is welcome to log in as user "guest", with password "guest_password".

Please feel free to join the Database Wiki project mailing list.
